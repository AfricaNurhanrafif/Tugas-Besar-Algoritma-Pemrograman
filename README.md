# Tugas-Besar-Algoritma-Pemrograman
APLIKASI MANAJEMEN TUGAS RUMAH TANGGA

Aplikasi ini digunakan untuk membantu pengguna dalam mengelola tugas-tugas rumah tangga sehari-hari. Data utama yang digunakan adalah daftar tugas rumah tangga yang berisi nama tugas, kategori, status pengerjaan, dan tingkat prioritas. Pengguna aplikasi adalah individu atau keluarga yang ingin mengatur dan memantau pekerjaan rumah agar lebih terorganisir dan efisien.

Dengan Spesifikasi Berikut:
Pengguna dapat menambahkan, mengubah, dan menghapus tugas rumah tangga dalam daftar tugas yang tersedia.
Pengguna dapat mengubah status tugas dari Belum Selesai menjadi Selesai untuk memantau progres pekerjaan rumah.
Sistem menyimpan dan menampilkan seluruh daftar tugas beserta informasi kategori, status, dan prioritas tugas.
Pengguna dapat mencari tugas berdasarkan nama menggunakan Binary Search dan berdasarkan kategori menggunakan Sequential Search.
Pengguna dapat mengurutkan daftar tugas berdasarkan tingkat prioritas menggunakan Selection Sort (prioritas terendah ke tertinggi) dan Insertion Sort (prioritas tertinggi ke terendah).
